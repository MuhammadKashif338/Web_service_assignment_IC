# Simple Web Services with Node.js

This project provides two simple web services implemented using Node.js and Express.js. The services include:

1. **Palindrome Checker**: A service that checks if the provided string is a palindrome.
2. **Fibonacci Sequence Generator**: A service that generates a Fibonacci sequence up to a given number.

Additionally, a frontend (HTML, CSS, and JavaScript) is provided to interact with these services through a clean and modern interface.

## Prerequisites

To run this project, you need to have the following installed:

- **Node.js**: [Download and install Node.js](https://nodejs.org/)
- **npm** (Node Package Manager): npm comes bundled with Node.js.
